# Pygame Space Mission

Project implementing a Spaceship Simulator using Pygame.

## Features implemented
- **Basic Functionality**:
    - Spaceship movement (Left/Right) within bounds.
    - Random meteorite spawning.
    - Shooting mechanics.
    - Game Over on collision (Hull depletion).
- **Extended Functionality**:
    - Starfield background animation.
- **Class Structure**:
    - `SpaceShip`: Manages fuel, hull, speed.
    - `CrewMember`: Defined with roles and health.
    - `Mission`: Tracks score and game status.

## How to run
1. Install requirements: `pip install -r requirements.txt`
2. Run game: `python main.py`
